### 小测验
输入三个数，判断是等边等腰还是其他
```cpp
int func(int x, int y, int z)
{
    if(x<=0 || y<=0 || z<=0)
    {
        return 0;
    }
    if(x+y-z<=0 || x+z-y<=0 || y+z<=0)
    {
        return 0;
    }
    if(x==y && x==z)
    {
        return 1;
    }
    if(x==y || y==z || z==x)
    {
        return 2;
    }
    return 3;
}
```

### 测试七大原则
- 测试显示缺陷存在，但不能证明系统不存在缺陷
- 穷尽测试是不可能的，应设置及时终止的条件
